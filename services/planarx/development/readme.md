# Planarx Civic Planning Tool (`arx-planarx`)

GIS-enabled civic planning platform for proposal management, voting, crowdfunding, and community engagement.

## Overview
`arx-planarx` delivers unique GIS integration, user roles, and workflows for civic planning that extend beyond core BIM functionality.

## Features
- GIS-based planning and visualization
- Proposal lifecycle management and voting
- Community crowdfunding and engagement tools
- Custom user roles for planners, officials, and the public
