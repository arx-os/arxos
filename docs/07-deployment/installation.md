# Physical Installation Guide

## Mounting and Wiring Nodes

### Installation Locations

#### Electrical Panels
```
[Panel] ─── [Node]
   │          │
   CT ───────┘
   
Current transformer clamps on mains
Node inside or adjacent to panel
```

#### Outlet Nodes
```
Remove outlet → Install node behind → Reconnect outlet
Node fits in standard electrical box
```

#### Ceiling Mount
```
Drop ceiling: Above tiles
Hard ceiling: Surface mount box
Optimal height: 8-10 feet
```

### Wiring Diagrams

```
AC Power Connection:
L (Hot) ───┬─── AC/DC ─── Node
N (Neutral) ┘      5V
```

### Safety Requirements

1. **Turn off breaker** before installation
2. **Test with meter** before touching
3. **Use proper PPE**
4. **Follow local codes**
5. **Document everything**
