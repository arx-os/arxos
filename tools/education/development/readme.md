# Arx Education (`arx-education`)

Course content, interactive logic simulations, and ArxIDE training modules for trade education and platform onboarding.

## Overview
`arx-education` is the primary source for internal and external training materials, interactive modules, and animated logic simulations for the Arxos platform.

## Features
- Self-paced courses and certifications
- Animated SVGX logic simulations
- ArxIDE onboarding guides and exercises
- Trade-specific modules for electrical, HVAC, plumbing, and more

## Usage
- Access modules through ArxIDE and platform onboarding flows
- Contribute new content using standardized templates from [arx-docs](https://github.com/capstanistan/arx-docs)

## Contribution
- Follow onboarding and style guidelines from [arx-docs](https://github.com/capstanistan/arx-docs)
- All new modules require review and cross-linking
