Get Started in Minutes

Start with ArxOS Core for free, upgrade as you grow

1

Download ArxOS Core

Get the free building operating system

curl -sSL https://install.arxos.dev | bash
2

Create Your First Building

Model your building with spatial data and equipment

arxos building create "My Building"
3

Add IoT Devices

Connect sensors and actuators from our certified marketplace

arxos device add --marketplace
4

Automate Workflows

Create visual automation workflows with n8n integration

arxos workflow create --template
