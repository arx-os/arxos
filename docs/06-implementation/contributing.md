# Contributing to Arxos

## Join the Building Revolution

### Getting Started

1. **Fork** the repository
2. **Clone** your fork
3. **Build** the project
4. **Make** changes
5. **Test** thoroughly
6. **Submit** PR

### Contribution Areas

#### Protocol Development
- Optimize 13-byte structure
- New object types
- Compression algorithms

#### Hardware Design
- New node types
- PCB layouts
- 3D printed cases

#### Terminal Interface
- ASCII art improvements
- New visualizations
- Game mechanics

#### Documentation
- Tutorials
- Translations
- Video guides

### Code Style

```rust
// Use rustfmt
cargo fmt --all

// Use clippy
cargo clippy --all-targets

// Document everything
/// Represents a building object in 13 bytes
pub struct ArxObject { ... }
```
