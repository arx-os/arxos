# Arx Desktop App (`arx-desktop`)

Electron or native desktop app for facility managers, enabling offline-first workflows, bulk edits, and object export.

## Overview
`arx-desktop` is designed for larger clients and facility managers who require robust offline access, bulk editing capabilities, and seamless export of building data.

## Features
- Full offline operation with local data storage
- Bulk editing and import/export of building models
- Direct integration with ArxLink and CMMS modules
- Intuitive desktop UI for non-technical users
