# Arx Data Vendor Tools (`arx-data-vendor`)

Bulk data export, import, and offline management tools for facility managers and data vendors.

## Overview
`arx-data-vendor` offers specialized tools for bulk editing, exporting, and synchronizing building datasets, tailored for data vendors and large-scale facility management.

## Features
- Bulk object export/import (CSV, JSON, SVGX)
- Offline-first workflows for large datasets
- Integration with desktop and CMMS modules
- Audit trails and role-based permissions
