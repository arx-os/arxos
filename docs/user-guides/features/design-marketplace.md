# ArxIDE Design Marketplace User Guide

## Overview

The ArxIDE Design Marketplace is a decentralized creator platform that allows architects, engineers, and builders to publish, license, and sell building and system designs authored in SVGX format. This guide provides step-by-step instructions for using all marketplace features.

## Getting Started

### Creating Your Account

1. **Sign Up for ArxIDE**
   - Download and install ArxIDE
   - Create your Arxos account
   - Complete your profile with professional information

2. **Link Your BILT Wallet**
   - Navigate to Settings â†’ Wallet
   - Create or import your BILT wallet
   - Verify your wallet address
   - Set up two-factor authentication for security

3. **Verify Your Creator Status**
   - Submit professional credentials
   - Complete identity verification
   - Wait for approval (typically 24-48 hours)

### Understanding the Interface

The marketplace is integrated directly into ArxIDE with these main sections:

- **Browse**: Discover and search for designs
- **Publish**: Upload and manage your designs
- **Dashboard**: Track your sales and analytics
- **Wallet**: Manage your BILT transactions

## Publishing Your Designs

### Preparing Your Design

1. **Create Your SVGX Design**
   - Use ArxIDE to create your building or system design
   - Ensure all components are properly tagged
   - Test your design for compatibility
   - Add comprehensive documentation

2. **Design Requirements**
   - Must be in valid SVGX format
   - Include all necessary dependencies
   - Provide clear documentation
   - Add appropriate tags and categories

### Publishing Process

1. **Access the Publishing Tool**
   - Open your design in ArxIDE
   - Click "Publish to Marketplace"
   - Or use the standalone publishing interface

2. **Fill Design Information**
   ```
   Title: [Your design name]
   Description: [Detailed description]
   Category: [Select primary category]
   Tags: [Add relevant tags]
   License Type: [Choose license]
   Price: [Set ARX price]
   ```

3. **Set Pricing and Licensing**
   - **Royalty-Free**: One-time purchase, unlimited use
   - **One-Time Commercial**: Single commercial use
   - **Open Source**: Free with attribution
   - **Subscription**: Recurring access

4. **Generate Preview**
   - System automatically creates preview images
   - Review and adjust if needed
   - Add additional preview images

5. **Publish Design**
   - Review all information
   - Confirm publication
   - Design is minted on blockchain
   - Available in marketplace immediately

### Design Management

#### **Updating Your Design**
1. Open your design in ArxIDE
2. Make your changes
3. Click "Update Design"
4. Set new version number
5. Update changelog
6. Confirm update

#### **Managing Versions**
- Each design maintains version history
- Users can access previous versions
- Version control is Git-style
- Branching and merging supported

## Discovering and Purchasing Designs

### Browsing the Marketplace

1. **Access the Marketplace**
   - Open ArxIDE
   - Click "Marketplace" in the sidebar
   - Or use the web interface

2. **Search and Filter**
   - Use the search bar for keywords
   - Filter by category, price, tags
   - Sort by popularity, rating, date
   - Filter by verified creators only

3. **Design Categories**
   - **K-12**: Educational building designs
   - **Residential**: Home and apartment designs
   - **MEP**: Mechanical, electrical, plumbing
   - **Interiors**: Interior design and furnishings
   - **BAS**: Building automation systems

### Design Details

When viewing a design, you'll see:

- **Design Preview**: Interactive 3D preview
- **Creator Information**: Name, verification status, rating
- **Design Details**: Size, complexity, compatibility
- **Reviews and Ratings**: User feedback
- **License Information**: Usage rights and restrictions
- **Price**: ARX amount and USD equivalent

### Purchasing a Design

1. **Review Design**
   - Examine the preview thoroughly
   - Read all documentation
   - Check compatibility with your project
   - Review license terms

2. **Select License Type**
   - Choose appropriate license for your use case
   - Understand usage restrictions
   - Consider future needs

3. **Complete Purchase**
   - Confirm ARX amount
   - Approve transaction in your wallet
   - Wait for blockchain confirmation
   - Download design files

4. **Access Your Purchase**
   - Design appears in your library
   - Import directly into ArxIDE
   - Access documentation and support

## Troubleshooting

### Common Issues

#### **Publishing Problems**
- **Design Validation Errors**: Check SVGX format and completeness
- **Upload Failures**: Verify file size and network connection
- **Pricing Issues**: Ensure ARX amount is within limits
- **Preview Generation**: Wait for processing to complete

#### **Purchase Issues**
- **Transaction Failures**: Check wallet balance and gas fees
- **Download Problems**: Verify purchase confirmation
- **Import Errors**: Check ArxIDE compatibility
- **License Confusion**: Review license terms carefully

#### **Technical Problems**
- **Slow Performance**: Check internet connection
- **Display Issues**: Update ArxIDE to latest version
- **Wallet Connection**: Reconnect wallet if needed
- **Search Problems**: Try different search terms
