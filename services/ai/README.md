# Arx AI Services (`arx-ai-services`)

Modular microservices for GPT-powered logic, geometry validation, voice input, AI agents, and more.

## Overview
`arx-ai-services` contains sidecar microservices that provide advanced AI/ML capabilities to the Arxos platform in a scalable, decoupled fashion.

## Features
- GPT-based logic and compliance checking
- Geometry validation and automated corrections
- Voice input and transcription services
- Extensible agent framework for automation
